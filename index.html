How to get your Hugging Face API Key

  1. Go to huggingface.co/settings/tokens
  2. Click New token (top right)
  3. Give your token a name (e.g., "FitMyResume")
  4. Check Make calls to Inference Providers under Inference
  5. Click Create and copy your token
  6. Paste it in the field above

Never share your API key publicly.
